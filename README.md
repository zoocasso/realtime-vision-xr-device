# 🥽 XR Device Hackathon: 실시간 시각 개선 솔루션
> **2024 판교 XR Device 메이커톤 (Sole Developer)**

안개 및 저조도 등 열악한 환경에서도 XR 기기를 통해 선명한 시야를 확보할 수 있도록 돕는 **실시간 영상 보정 및 시각 극복 장치** 개발 프로젝트입니다.

---

## 📋 프로젝트 개요 (Project Overview)
본 프로젝트는 XR(Extended Reality) 기술을 활용하여 재난 현장, 안개 자욱한 도로, 혹은 야간 상황에서 사용자의 시각적 한계를 극복하는 것을 목표로 합니다. 실시간 영상 처리 알고리즘을 통해 시야를 방해하는 요소를 제거하고 가시성을 극대화하는 하드웨어 및 소프트웨어 통합 솔루션을 제안했습니다.

* **주제:** 실시간 안개 제거(Dehazing) 및 저조도(Low-light) 환경 극복 장치
* **행사:** 2024 판교 XR Device 메이커톤 (2024 Pangyo XR Device Makerthon)
* **역할:** 1인 개발 (기획, 알고리즘 구현, 하드웨어 구성, 발표)

## ✨ 주요 기능 (Key Features)
1. **실시간 안개 제거 (Real-time Dehazing):** 대기 중 산란된 빛의 수치를 계산하여 안개로 흐려진 영상을 선명하게 복원합니다.
2. **어두운 환경 극복 (Low-light Enhancement):** 저조도 환경에서 노이즈를 억제하고 밝기를 확보하여 사물 식별 능력을 향상시킵니다.
3. **XR 최적화:** 착용형 기기에서 지연 시간(Latency)을 최소화하여 이질감 없는 실시간 시각 피드백을 제공합니다.

* **Library:** (예: OpenCV, Unity)
* **Hardware:** XR Device, (예: Metalens2)
